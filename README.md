# Java Console App – Product Manager

Mini applicazione console sviluppata in Java per la gestione di prodotti.
Il programma permette di inserire prodotti, visualizzarli, calcolare statistiche e rimuovere elementi dalla lista.

Questo progetto è stato realizzato per consolidare:
- Programmazione ad Oggetti (OOP)
- Utilizzo di ArrayList
- Cicli (for-each, do-while)
- Validazione input
- Calcolo di statistiche (somma, media, massimo, minimo)

## Funzionalità

- Inserimento prodotto (nome + prezzo)
- Validazione prezzo (> 0)
- Visualizzazione elenco prodotti
- Calcolo:
  - Somma totale prezzi
  - Media prezzi
  - Prodotto con prezzo massimo
  - Prodotto con prezzo minimo
- Rimozione ultimo prodotto inserito
- Menu interattivo con do-while


## Concetti utilizzati

- Incapsulamento
- Costruttori
- Getter
- ArrayList (add, get, size, remove)
- For-each
- Gestione logica del flusso

## Obiettivo del progetto

Esercizio per migliorare la gestione dinamica di collezioni di oggetti in Java.
